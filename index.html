Xrafts Contents Editor

XCE AI — the AI-native workspace for playable content.

It is not a traditional game editor, not a single-format generation tool, and not another traffic-heavy closed platform. It is a lightweight interactive creation workbench for turning stories, characters, scenes, gameplay and expression into playable, publishable and continuously iterated digital works.

C++ editor · no-code · visual events · TypeScript API · multi-platform publishing

AI is not bolted on. It is inside the creation chain.

XCE AI uses AI to generate and organize 2D/3D assets, characters, actions, CG animation, effects, music, voice, game logic, automated tests, tuning suggestions and feedback analysis.

  • Generate editable assets into the project asset system.
  • Transform natural language into XCE AI DSL, events and components.
  • Use AI player agents to test flows, interactions and balance.
  • Ship playable works instead of disconnected generated files.
